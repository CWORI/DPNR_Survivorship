All Coral Cover

Overview

This section contains whole-plot coral cover results for all coral annotations, not just tracked outplants. Every plot uses the same workflow and appears on this page as soon as its survey workbooks are added to data_raw/All_Coral/ and its plot config is completed under config/all_coral_survey_files/.

The current practice dataset is Plot E. Plots A through H will remain together in this one All Coral Cover section as more surveys are added.

Adding Another Plot

  1. Put the reviewed Excel workbooks under data_raw/All_Coral/Plot_<letter>/<year>/.
  2. Add one row per survey to config/all_coral_survey_files/Plot_<letter>.csv.
  3. Render the Quarto website.
  4. Check the file audit and QA/QC tables before publishing.

No new Quarto page or navigation item is needed for later plots; they will join this section automatically.
